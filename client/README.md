# Attendance Management System - Client

React frontend for the School Attendance Management System.

## Setup

```bash
npm install
npm start
```

Runs on http://localhost:3000. Proxies API calls to http://localhost:5000.

## Features
- Dashboard with attendance overview
- Mark attendance (present/absent/late/excused)
- Student management (add, edit, delete)
- Class management
- Reports and analytics with sorting
